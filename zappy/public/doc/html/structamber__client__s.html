Detailed Description

Structure representing an Amber client.

This structure contains the necessary information to handle an Amber client. It includes a TCP connection object.

Parameters
_tcpThe TCP connection object for the client.
_bufferA buffer to store the data received from the client.
_team_nameThe name of the team the client belongs to.
_directionThe direction the client is facing.
_xThe x-coordinate of the client.
_yThe y-coordinate of the client.
_levelThe level of the client.
_idThe ID of the client.
_queue_commandThe queue of commands to be executed by the client.
_inventoryThe inventory of the client.
_elapsed_timeThe elapsed time since the last command.
_is_graphicalA boolean flag indicating if the client is graphical.
_is_incantatingA boolean flag indicating if the client is incantating.
