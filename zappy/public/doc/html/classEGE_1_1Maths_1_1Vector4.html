Detailed Description

template<typename T>
class EGE::Maths::Vector4< T >

A 4D vector class template.

This class represents a 4D vector with x, y, z and w components.

Template Parameters
TThe type of the components of the vector.
Note
The template parameter T must be an arithmetic type.
