\doxysubsection{Detailed Description}
The \doxylink{classOnyx_1_1MainMenu}{Main\+Menu} class represents the main menu of the application. 

The \doxylink{classOnyx_1_1MainMenu}{Main\+Menu} class is responsible for managing the main menu of the application. It provides functions for updating and looping the menu, checking if the application is running, creating the main panel, and getting the TCP client. It also contains member variables for storing the running state, TCP client, window, camera, delta time, and user interface.
