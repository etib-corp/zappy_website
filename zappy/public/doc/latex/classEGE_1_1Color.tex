\doxysubsection{Detailed Description}
Represents an RGBA color. 

The \doxylink{classEGE_1_1Color}{Color} class provides a convenient way to represent colors in RGBA format. Each color component (red, green, blue, alpha) is represented by an unsigned char value ranging from 0.\+0f to 1.\+0f.
