\doxysubsection{Detailed Description}
Represents an event in the game engine. 

The \doxylink{classEGE_1_1Event}{Event} class provides functionality to handle various types of events, such as keyboard, mouse, window, joystick, and trigger events. It allows registering triggers and callbacks to be executed when specific events occur.
