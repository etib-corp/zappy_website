\doxysubsection{Detailed Description}
The base class for interactable \doxylink{classEGE_1_1GUI}{GUI} elements. 

This class provides a common interface for \doxylink{classEGE_1_1GUI}{GUI} elements that can be interacted with, such as buttons. It inherits from the \doxylink{classEGE_1_1GUI}{GUI} class.
