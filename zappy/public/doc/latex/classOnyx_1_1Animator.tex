\doxysubsection{Detailed Description}
The \doxylink{classOnyx_1_1Animator}{Animator} class is responsible for managing animations for a player. 

The \doxylink{classOnyx_1_1Animator}{Animator} class provides methods to start and update animations for a player. It supports different types of animations and allows setting the time unit for animation updates.
