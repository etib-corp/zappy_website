Detailed Description

Represents a singleton callback function for handling file drop events in a GLFW window.

The GlfwWindowFileDropCallbackFunction class provides a singleton instance of a callback function that can be used to handle file drop events in a GLFW window. It ensures that only one instance of the callback function exists at a time.
