\doxysubsection{Detailed Description}
Represents a menu in the \doxylink{classEGE_1_1GUI}{GUI}. 

This class is a derived class of the \doxylink{classEGE_1_1Container}{Container} class and represents a menu in the graphical user interface (\doxylink{classEGE_1_1GUI}{GUI}). It provides functionality to draw the menu.
