A class for representing a matrix. 

The \doxylink{classEGE_1_1Maths_1_1Matrix}{Matrix} class represents a matrix with the specified width and height. 
\begin{DoxyTemplParams}{Template Parameters}
{\em WIDTH} & The width of the matrix. \\
\hline
{\em HEIGHT} & The height of the matrix. \\
\hline
{\em T} & The type of the elements in the matrix. \\
\hline
\end{DoxyTemplParams}
\begin{DoxyNote}{Note}
The width and height must be greater than 0. 

The type T must be an arithmetic type. 
\end{DoxyNote}
